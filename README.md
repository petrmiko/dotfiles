# Dotfiles

Collection of personal dotfiles and install scripts.

## MacOS
Run in terminal
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/petrmiko/dotfiles/master/install-mac.sh)"
```

## Fedora
Run in terminal
```bash
git clone https://github.com/petrmiko/dotfiles.git $HOME/.dotfiles
cd $HOME/.dotfiles
./install-fedora.sh
```
